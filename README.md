# bigdoodr.github.io

Personal website for Casey Scruggs — Mac admin, app developer, and tinkerer.

## What's here

**Slash pages** — A collection of `/slashes`-style pages covering what I use, read, watch, believe, and build. Highlights include `/uses`, `/top4`, `/shoebox`, `/follows`, and more.

**Blog** — Occasional posts on Mac administration, app development, and whatever else is on my mind.

**Exercise utilities** — Web tools I built to support my fitness goals, some of which grew into iOS apps:
- [Bodyweight WorkoutRandomizer](https://testflight.apple.com/join/FfJyugX3) (TestFlight)
- [ExpandableExerciseTimer](https://apps.apple.com/us/app/expandableexercisetimer/id6756544631) (App Store)

**Presentations** — Slides and resources from conference talks, including MacAdmins and WKU Pro Dev Day.

**Bookmarks** — A curated, categorized collection of links I actually use.
