Use this terminal command on your Mac to generate a file list for your GitHub repository:

find . -type f -not -path "./.git/*" -not -name ".DS_Store" | head -50 | sort > sitemap-files.txt

Or use this enhanced version that includes file types and sizes:

find . -type f -not -path "./.git/*" -not -name ".DS_Store" -exec ls -la {} \; | awk '{print $9 " (" $5 " bytes)"}'| head -50 | sort > detailed-sitemap.txt
