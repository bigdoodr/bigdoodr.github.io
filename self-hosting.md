Services and tools I run and manage myself, because owning your own stuff matters.

## 🏠 Home Server / NAS

I run a small home server that hosts my media management stack, accessible remotely via a [Cloudflare Tunnel](https://www.cloudflare.com/products/tunnel/).

- **[Jellyfin](https://jellyfin.org)** — My personal media server. Movies and TV, streamed on my terms, with no subscription required.
- **[Radarr](https://radarr.video)** — Automated movie collection management. Keeps my library organized and up to date.
- **[Sonarr](https://sonarr.tv)** — Same idea as Radarr, but for TV shows.
- **[Notifiarr](https://notifiarr.com)** — Notification and integration hub that ties the rest of the stack together.

## 📓 Obsidian

I use [Obsidian](https://obsidian.md) to track my exercise routines and personal notes. My vault lives in iCloud, so I'm managing the data and the setup — iCloud just handles the syncing across devices.
